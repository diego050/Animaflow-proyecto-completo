### Ingestion Results
- **Icons downloaded:** 43,280 from 10 collections (mdi, tabler, material-symbols, fa, lucide, heroicons, simple-icons, ph, bi, ion).
- **Embeddings generated:** 43,280 processed in ~10 minutes.
- **Database:** 43,237 icons inserted (43 duplicates skipped).

### How it works
1. LLM generates `media_query` for scene.
2. Backend calls `find_best_icons(media_query)` → pgvector search → returns top 5 relevant icons.
3. Icons injected into LLM prompt as suggestions.
4. LLM chooses best icon using `type: "component", componentName: "IconifyIcon", icon: "mdi:ecg-heart"`.
5. Frontend renders via `https://api.iconify.design/{prefix}/{name}.svg`.

## 5. Production Deployment Notes
- Run ingestion via Docker container (isolated, auto-cleanup):
  ```bash
  docker run --rm --network animaflow_animaflow_net \
    --dns 8.8.8.8 --dns 1.1.1.1 \
    -v $(pwd)/scripts:/scripts \
    -e DATABASE_URL="postgresql://postgres:postgres@postgres:5432/animaflow" \
    python:3.11-slim bash -c "
      pip install --no-cache-dir httpx sentence-transformers psycopg2-binary sqlalchemy &&
      python /scripts/download_iconify_index.py &&
      python /scripts/generate_embeddings.py
    "
  ```
- No additional services added to docker-compose (keeps 4 services).
- Runtime uses existing Gemini API (no PyTorch in production image).
