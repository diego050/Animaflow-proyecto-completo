# Habilidades Técnicas de AnimaFlow (Technical Skills)

## Core Stack y Lenguajes Frontend
* **Mapeo:** Frontend Agent / Architecture Agent
* **Habilidades Asignadas:**
  * Desarrollo web de interfaces robustas utilizando **React** y transpiladores ágiles como **Vite**.
  * Arquitectura e implementación estricta de dominios mediante tipos utilizando **TypeScript**.
  * Manejo del sistema de utilidades y estilos ágiles usando **TailwindCSS**.
  * Administración avanzada del árbol de estados y flujos funcionales con **Zustand** o **Redux**.

## Tecnologías de Video e Inteligencia Artificial
* **Mapeo:** Frontend Agent / Backend Agent / Architecture Agent
* **Habilidades Asignadas:**
  * Renderizado server-side de recursos en canvas HTML usando **Remotion**.
  * Cálculo algorítmico para garantizar una fidelidad posicional estricta y cronometrada (Sync frame-accurate).
  * Prompteado estructurado e integraciones con **Large Language Models (LLMs)** para la partición contextual de guiones y textos semánticos.
  * Consumo e integración de redes neuronales cloud de Text-To-Speech (**TTS Cloud**) con especial atención a ElevenLabs y PlayHT.

## Sistemas de Backend y Base de Datos
* **Mapeo:** Backend Agent / Architecture Agent
* **Habilidades Asignadas:**
  * Implementación y arquitectura de APIs concurrentes de alta velocidad utilizando **FastAPI** en el ecosistema **Python**.
  * Verificación robusta de serialización de datos de la web hacia memoria mediante **Pydantic**.
  * Diseño de modelado relacional e indexación sobre motores transaccionales **PostgreSQL**.
  * Acoplamiento seguro entre entidades relacionales y orientadas a objetos utilizando ORMs avanzados como **Prisma** o **SQLAlchemy**.
  * Especificación algorítmica y arquitectónica de JSON Serializado Avanzado (**spec.json Schema Design**).

## Infraestructura, Cola de Mensajes y Operaciones (DevOps)
* **Mapeo:** Backend Agent / QA Agent / Architecture Agent
* **Habilidades Asignadas:**
  * Desacoplamiento de flujos bloqueantes mediante arquitecturas de publicador-suscriptor usando workers con **Celery** o **RQ**.
  * Diseño de infraestructura en la nube y despliegue rápido utilizando **Vercel** o **Hostinger**.
  * Gestión de buckets de almacenamiento persistente de recursos mediante **AWS S3** o **Cloudinary**.
  * Configuración y optimización de rutinas de integración y entrega continua (CI/CD) empleando **GitHub Actions**.
  * Autenticación moderna, criptografía de tokens (JWT) y manejo de RBAC con **Clerk** o **Supabase Auth**.

## Testing y Frameworks de Pruebas
* **Mapeo:** QA Agent
* **Habilidades Asignadas:**
  * Desarrollo asertivo de test suites de integración y unitarias usando **Jest** y **Pytest**.
  * Simulación completa de usuario automatizada end-to-end con **Playwright**.
